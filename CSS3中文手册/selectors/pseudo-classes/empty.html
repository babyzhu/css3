伪类选择符 CSS Pseudo-Classes Selectors

E:empty

语法：

E:empty{srules}

说明：

匹配没有任何子元素（包括text节点）的元素E。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
