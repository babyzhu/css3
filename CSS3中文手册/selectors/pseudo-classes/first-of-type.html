伪类选择符 CSS Pseudo-Classes Selectors

E:first-of-type

语法：

E:first-of-type{srules}

说明：

匹配同类型中的第一个同级兄弟元素E。
要使该属性生效，E元素必须是某个元素的子元素，E的父元素最高是body，即E可以是body的子元素
该选择符总是能命中父元素的第1个为E的子元素，不论第1个子元素是否为E

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
