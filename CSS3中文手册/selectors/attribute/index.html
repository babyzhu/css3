选择符 css Selectors Module

属性选择符 Attribute Selectors

选择符 版本 描述
E[att] CSS2 选择具有att属性的E元素。
E[att="val"] CSS2 选择具有att属性且属性值等于val的E元素。
E[att~="val"] CSS2 选择具有att属性且属性值为一用空格分隔的字词列表，其中一个等于val的E元素。
E[att^="val"] CSS3 选择具有att属性且属性值为以val开头的字符串的E元素。
E[att$="val"] CSS3 选择具有att属性且属性值为以val结尾的字符串的E元素。
E[att*="val"] CSS3 选择具有att属性且属性值为包含val的字符串的E元素。
E[att|="val"] CSS2 选择具有att属性且属性值为以val开头并用连接符"-"分隔的字符串的E元素，如果属性值仅为val，也将被选择。

以上选择符的兼容性都是一样的：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持

E[att]

语法：

E[att]{srules}

说明：

选择具有att属性的E元素。

事例： 
							img[alt]{border:solid 1px #F00}
	
							<img src="xxx" alt="" />
							<img src="xxx" />

第一张图会加一个红色边框，因为匹配到了alt属性

E[att="val"]

语法：

E[att="val"]{srules}

说明：

选择具有att属性且属性值等于val的E元素。

事例： 
							input[type="text"]{border:solid 1px #F00}
	
							<input type="text" />
							<input type="submit" />

此例，将会命中第一张input，因为匹配到了alt属性，并且属性值为text

E[att~="val"]

语法：

E[att~="val"] { sRules }

说明：

选择具有att属性且属性值为一用空格分隔的字词列表，其中一个等于val的E元素（包含只有一个值且该值等于val的情况）。

事例： 
							div[class~="a"]{border:slide 1px #f00}
							
							 <div class="a">1</div>
							 <div class="b">2</div>
							 <div class="a b">3</div>

此例，将会命中1, 3两个div，因为匹配到了class属性，且属性值中有一个值为a

E[att^="val"]

语法：

E[att^="val"] { sRules }

说明：

选择具有att属性且属性值为以val开头的字符串的E元素。

事例： 
							div[class^="a"] {border:slide 1px #f00}
							<div class="abc">1</div>
							<div class="acb">2</div>
							<div class="bac">3</div>

此例，将会命中1, 2两个div，因为匹配到了class属性，且属性值以a开头

E[att$="val"]

语法：

E[att$="val"] { sRules }

说明：

选择具有att属性且属性值为以val结尾的字符串的E元素。

事例： 
							div[class$="c"] {border:slide 1px #f00}
							<div class="abc">1</div>
							<div class="acb">2</div>
							<div class="bac">3</div>

此例，将会命中1, 3两个div，因为匹配到了class属性，且属性值以c结尾

E[att*="val"]

语法：

E[att*="val"] { sRules }

说明：

选择具有att属性且属性值为包含val的字符串的E元素。

事例： 
							div[class*="c"] {border:slide 1px #f00}
							<div class="abc">1</div>
							<div class="acb">2</div>
							<div class="bac">3</div>

此例，将会命中所有div，因为匹配到了class属性，且属性值中都包含了b

E[att|="val"]

语法：

E[att|="val"] { sRules }

说明：

选择具有att属性且属性值为以val开头并用连接符"-"分隔的字符串的E元素，如果属性值仅为val，也将被选择。

事例： 
							div[class|="a"] {border:slide 1px #f00}
							<div class="a-test">1</div>
							<div class="b-test">2</div>
							<div class="c-test">3</div>

此例，将会命中第一个div，因为匹配到了class属性，且属性值以紧跟着"-"的a开头
