flex-direction

语法：

flex-direction:取值

说明：

flex-direction 设置或检索伸缩盒对象的子元素在父容器中的位置

flex-direction属性取值
属性值 说明
row 横向从左到右排列（左对齐），默认的排列方式。(水平方向)
column 纵向排列（垂直方向）
row-reverse 反转横向排列（右对齐，从后往前排，最后一项排在最前面。
column-reverse 反转纵向排列，从后往前排，最后一项排在最上面。

使用方法：

flex-direction:row; /*弹性盒子子元素水平方向排列左对齐*/

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
较早版本 6.0-10.0 4.0-19.0 5.1.7-6.0 21.0-28.0-webkit- 9.5-12.0
较新版本 11.0 20.0 7.0-webkit- 29.0 12.1

橘色说明要加浏览器的前缀

事例：


	<ul id="box3">
		<li>a</li>
		<li>b</li>
		<li>c</li>
	</ul>

#box3{display:-webkit-flex;display:flex;width:220px;margin:0;padding:0;list-style:none;-webkit-flex-direction:row;flex-direction:row;}

#box3 li{width:100px;height:100px;border:1px solid #aaa;text-align:center;}

flex-direction:row-reverse
