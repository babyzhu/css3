order

语法：

order:整数值（可以负数）

说明：

order 设置或检索弹性盒模型对象的子元素出F的序。数值小的排在前面。可以为负数

使用方法：

order：-1; /**/

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
较早版本 6.0-10.0 4.0-19.0 5.1.7-6.0 21.0-28.0-webkit- 9.-12.0
较新版本 11.0 20.0 7.0-webkit- 29.0 12.1

橘色说明要加浏览器的前缀

事例：


	<ul id="box" class="box">
		<li>a</li>
		<li>b</li>
		<li>c</li>
		<li>d</li>
		<li>e</li>
	</ul>

.box{
	display:-webkit-flex;
	
	display:flex;
	
	margin:0;padding:10px;list-style:none;background-color:#eee;}
	
.box li{width:100px;height:100px;border:1px solid #aaa;text-align:center;}

#box li:nth-child(3){
	-webkit-order:-1;
	order:-1;
}
