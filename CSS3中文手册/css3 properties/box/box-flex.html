box-flex

语法：

box-flex:取值

说明：

box-flex属性定义盒子的弹性空间

使用方法：

box-flex:1.0; /**/

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
较早版本 6.0-11.0 4.0.-25.0 -moz- 5.1.7 -webkit- 4.0-31.0 -webkit- 9.5-14.0
较新版本 15.0-18.0 -webkit-

橘色说明要加浏览器的前缀

事例：

 body
        {
            display:-moz-box;
            -moz-box-orient:horizontal;   /*定义盒子元素内的元素从左到右流动显示*/
            width:200px;
            height:150px;
        }
        #box1
        {
            background:red;
            -moz-box-flex:1.0;
        }
        #box2
        {
            background:blue;
            -moz-box-flex:2.0;
        }
        #box3
        {
            background:orange;
            -moz-box-flex:1.0;
        }
