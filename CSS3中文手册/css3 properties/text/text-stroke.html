text-stroke

语法：

text-stroke:宽度值 颜色值;

说明：

text-stroke给文字添加边框；

注意：

text-stroke是一个复合属性，由text-stroke-width和text-stroke-color两个属性组成

text-stroke-width属性：设置描边的宽度，可以为一般的长度值

text-stroke-color属性：设置描边的颜色。

使用方法：

text-stroke:2px red;; /*给文字瞄了1像素的红色描边*/

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6-10 4-18 5.1.7 13 11.5-12.5

事例：

div{						
							text-stroke:1px red;
							 -webkit-text-stroke:1px red;
							 -moz-text-stroke:1px red;
							 -o-text-stroke:1px red;
							}
