word-wrap

语法：

说明：

word-wrap用来设置“长单词”或者“URL地址”是否要换行到下一行，normal是默认值

此属性不常用，用浏览器默认值即可。一般用在英文文档里

使用方法：

word-wrap：break-word; /*强制换行*/

兼容性：

所有的浏览器都支持

事例：

div{						
							width:200px;
							height:120px;
							border:1px solid gray;
							}
						

div{						
							width:200px;
							height:120px;
							border:1px solid gray;
							word-wrap:break-word;
							}
