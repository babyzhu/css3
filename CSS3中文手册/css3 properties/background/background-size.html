background-size

语法：

background-size:auto | 长度值 | 百分比 | cover | contain

说明：

background-origin:设置背景图片的大小，以长度值或者百分比显示，还可以通过cover 和contain来对图片进行伸缩

语法中的参数：

1.auto:默认值，不改变背景图片的原始高度和宽度

2.<长度值>：成对出现，比如200px 50px，将背景图片的高度和宽度依次设置为前面两个值，当设置一个值时，将其作为图片宽度来等比缩放；

3.百分比：0%~100%之间的任何值，将背景宽高依次设置为所在元素宽高乘以百分比得出来的数值，当设置一个值时 同上面

4.cover：顾名思义为覆盖，即背景图片等比例缩放以填满整个容器

5.contain：容纳，即将背景图片等比例缩放至某一边紧贴容器边缘为止

使用方法：

div{
							background:#ccc url(...) no-repeat;
							background-size:cover;
							}

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持

事例：

div{
						     background: url(images/1.png) no-repeat;/*images 的图片大小是100*60*/
							width: 120px;
							height: 120px;
							border: 1px solid #f00;
							background-size:auto;
							}
