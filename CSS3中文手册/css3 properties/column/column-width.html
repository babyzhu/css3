column-width

语法：

column-width:auto/长度值

说明：

column-count属性定义多列布局中每一列的宽度

column-width属性取值
属性值 说明
auto 默认值，根据column-count属性值自动分配宽度
长度值 可以为px、em或者百分比等

使用方法：

column-width:150px; /*设置列宽为150px*/

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6-9 4-18 5.1.7 13-23 11.1
版本 10

橘色说明要加浏览器的前缀

事例：

均分3列 
							body{
								widht:400px;
								padding:10px;
								border:1px solid #ccc;
								column-width:150px;
								-weibkit-column-width:150px;
								-moz-column-width:150px;
								-o-column-width:150px;
							}

这里使用“width:400px”限定了body宽度为400px，然后使用“column-width:150px;”定义列宽为150px，这样body就会自动根据容器宽度、列宽以及内容多少来计算列数。
