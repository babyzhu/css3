column-rule

语法：

column-rule:边框宽度 边框样式 边框颜色

说明：

column-rule属性定义列和列之间的边框样式，其中边框样式包括：宽度、颜色、样式

column-rule属性类是一个复合属性，由3个子属性组成

column-rule属性取值
属性值 说明
column-rule-width 设置边框的宽度
column-rule-style 设置边框的样式
column-rule-color 设置边框的颜色

使用方法：

column-rule:1px dashed red; /*列间边框为“1px的红色虚线”*/

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6-9 4-18 5.1.7 13-23 11.1
版本 10

橘色说明要加浏览器的前缀

事例：

列间边框为“1px的红色虚线” 
							body{
								width:400px;
								padding:10px;
								border:1px solid silver;
								-webkit-column-count:2;
								-webkit-column-gap:20px;    
								-webkit-column-rule:1px dashed red
							}
