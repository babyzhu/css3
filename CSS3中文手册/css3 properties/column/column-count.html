column-count

语法：

column-count:auto/正整数

说明：

column-count属性指定多列布局的列数

column-count属性取值
属性值 说明
auto 列数由其他属性决定，比如column-width
n（正整数） 元素内容被自动划分为最佳n列

使用方法：

column-count:3; /*均分成3列*/

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6-9 4-18 5.1.7 13-23 11.1
版本 10

橘色说明要加浏览器的前缀

事例：

均分3列 
							body{
								widht:400px;
								padding:10px;
								border:1px solid #ccc;
								column-count:3;
								-weibkit-column-count:3;
								-moz-column-count:3;
								-o-column-count:3;
							}
