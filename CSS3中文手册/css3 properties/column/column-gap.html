column-gap

语法：

column-gap:取值

说明：

column-gap属性定义列与列之间的间距

column-gap属性取值
属性值 说明
normal 浏览器默认的长度值
长度值 可以为px、em或者百分比等

使用方法：

column-gap:20px; /*列间距为20px*/

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6-9 4-18 5.1.7 13-23 11.1
版本 10

橘色说明要加浏览器的前缀

事例：

均分3列 
							body{
								width:400px;
								padding:10px;
								border:1px solid silver;
								-webkit-column-count:2;
								-webkit-column-gap:20px;    /*定义列间距为20px*/
							}
