opacity

语法：

opacity: 数值

说明：

opacity属性控制元素的透明度，取值范围0.0-1.0.0.0表示完全透明，1.0表示完全不透明，不可以为负数

使用方法：

opacity:.5; /*增加0.5的透明度*/

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
Values IE Firefox Chrome Safari Opera iOS Safari Android Browser Android Chrome
Basic Support 6.0-8.0 2.0+ 4.0+ 3.1+ 15.0+ 3.2+ 2.1+ 18.0+
9.0+

对于尚不支持opacity属性的IE浏览器可以使用IE私有的滤镜属性来实现与opacity相同的效果

事例：

a{
							display:inline-block;
							padding:5px 10px;
							font-family:微软雅黑;
							color:white;
							background-color:#45B823;
							border-radius:4px;
							cursor:pointer;
							}
							a:hover{
							filter:alpha(opacity=80);  /*lt IE8*/
							 opacity:.8;
							}
