min-height

说明：

我们在写CSS的时候，常常会遇到让一个图片或一个布局不能超出设定一定高度范围值

兼容写法

div{
	min-height:1000px;
	_height:expression((document.documentElement.clientHeight||document.body.clientHeight)>1000?"1000px":"");
}

这里的1000和1000px是你需要的数值，注意3个数值的相同。

IE6支持max-height又支持min-height方法 

div{
	max-height:620px;
	
	min-height:40px;
	
	_height:expression(this.scrollHeight > 1000 ? "1000px" : (this.scrollHeight < 100 ? "100px" : "auto"));
}

最大高度为1000px，最小高度为100px
