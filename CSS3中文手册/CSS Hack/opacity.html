opacity 兼容

opacity

语法：

opacity 兼容

说明：

设置文字或者背景的透明度(但在一个块中有文字和背景，设置透明值，两者皆起作用)

兼容写法

							opacity{filter:alpha(opacity=50)}/* IE */
							-moz-opacity:0.5;              /* 老版Mozilla */
							-khtml-opacity:0.5;              /* 老版Safari */
							opacity: 0.5;           /* 支持opacity的浏览器*/
